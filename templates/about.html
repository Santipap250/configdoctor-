เกี่ยวกับ OBIXConfig Doctor

OBIXConfig Doctor เป็นเครื่องมือที่ออกแบบมาเพื่อช่วยผู้เล่นโดรน FPV ให้เริ่มต้นตั้งค่า PID, Filter และ Preset ได้อย่างรวดเร็วและปลอดภัย โดยอาศัยแนวคิดและประสบการณ์จากนักบิน FPV ชุมชนไทย

จุดประสงค์

  • ช่วยให้ผู้ใช้มีค่าเริ่มต้น (baseline) ที่ปลอดภัยในการทดสอบบิน
  • ลดเวลาการตั้งค่าเริ่มต้นและความสับสนสำหรับมือใหม่
  • ให้เครื่องมือคัดลอก CLI เพื่อนำไปใช้กับ Betaflight ได้ทันที

ฟีเจอร์ v2.3 (19 เครื่องมือ)

  • Blackbox CSV Analyzer — วิเคราะห์ flight log · PID score A–D · motor imbalance
  • PID Symptom Advisor — 13 อาการ → วินิจฉัยอัตโนมัติ + CLI fix
  • CLI Surgeon — วิเคราะห์ diff all / dump · ตรวจ BF 4.3–4.5
  • RPM Filter Calculator — harmonic table + dyn_notch ครบ
  • ESC Compatibility Checker — ตรวจ amperage + DShot protocol
  • Rates Visualizer — ACTUAL / KISS / RC+ curve real-time
  • CLI Diff Comparator — เปรียบ 2 config ทันที
  • Motor × Prop Recommender — แนะนำ KV + stator ตามสเปค
  • VTX Suite — Bands, Range Calculator, SmartAudio
  • OSD Designer — ออกแบบ layout + export
  • DIFF / DUMP Library — ดาวน์โหลดไฟล์ตัวอย่างสำเร็จรูป
