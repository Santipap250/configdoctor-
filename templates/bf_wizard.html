📡 GYRO LOWPASS FILTER
Gyro LPF1 Type
PT1: latency ต่ำกว่า เหมาะ racing · BIQUAD: rolloff ชันกว่า เหมาะ freestyle
Gyro LPF1 (Hz)200 Hz
ต่ำ = filter หนัก latency มาก · สูง = fast response แต่ noise มากขึ้น

📉 D-TERM LOWPASS FILTER
D-term LPF1 Type
D-term LPF1 (Hz)110 Hz
ต่ำเกิน → D sluggish ไม่ทำงาน · สูงเกิน → motor ร้อน (D amplify noise)
D-term LPF2 (Hz) — 0 = ปิด0
⚡ RPM FILTER
RPM Filter — แนะนำอย่างยิ่ง
กรอง motor noise ที่แม่นยำที่สุด — ต้องการ Bidir DSHOT + รองรับ ESC
RPM Filter Harmonics3
3 harmonics = sweet spot · มากขึ้น = CPU load สูงขึ้น
RPM Filter Min Hz100
ควรต่ำกว่า idle RPM harmonic เล็กน้อย · ~100 Hz สำหรับ 5" motor

🛡️ FAILSAFE CONFIGURATION
Failsafe Action
DROP: ตัด motor ทันที (FPV racing/freestyle) · LAND: ลงช้าๆ (beginner) · GPS_RESCUE: บินกลับ home
Failsafe Delay (×0.1s)4
ค่า 4 = 0.4 วินาที · ต่ำเกิน → false trigger จาก interference · สูงเกิน → โดรนบินไปไกล
Failsafe Off Delay (×0.1s)1
เวลา motor ยัง spin หลัง failsafe trigger · 1 = ปลอดภัยสุด
⚙️ MOTOR SAFETY
Motor Stop

Motor Output Limit (%)100
100% = full power · 80–90% สำหรับ beginner หรือ บิน indoor
